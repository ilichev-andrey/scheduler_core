# SchedulerCore

SchedulerCore - это один из модулей проекта для автоматизации записи клиентов. 

## Функции

- Добавить пользователя
- Получить пользователя
- Получить работников
- Добавить услуги
- Получить список услуг
- Получить свободное время для записи
- ~~Записаться~~
- ~~Отменить запись~~
- ~~Получить историю посещений~~
